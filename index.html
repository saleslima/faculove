📄 Simulado automático por PDF

Gere questões com 4 alternativas, gabarito e nota final

Envie um PDF, escolha quantidade e dificuldade. A página extrai o texto, monta questões objetivas, salva no storage do navegador e corrige as respostas com comparação ao gabarito.

Storage do navegador
O PDF, o texto extraído, as questões, as alternativas, o gabarito e a última correção ficam salvos neste navegador usando IndexedDB.

Esta versão funciona localmente no navegador e não usa API externa. Ela gera as alternativas a partir dos próprios trechos do PDF. PDFs escaneados como imagem precisam de OCR para funcionar corretamente.
Para melhores resultados, use PDFs com texto selecionável, capítulos bem organizados e parágrafos completos.
